#タイマープラグイン

タイマー設定とそのリマインドが出来るプラグイン

##動作環境

Wordpress6.4.2/PHP8.2.0/mysql/管理画面内でのみ動作

##インストール方法

timerフォルダをpluginsフォルダに入れる
プラグインを有効化する

#使用されるオプション名

$user_login+"_dashbord_reminder"
$user_login+"_dashbord_timer"
$user_login:ログイン中のユーザー名

##操作方法

プラグインが有効化されると、ダッシュボードに設定用のUIが表示される
予定と時間を入力し確定を押す

再登録・ユーザー別の登録が可能

##その他

カピバラの画像と情報も表示されます
